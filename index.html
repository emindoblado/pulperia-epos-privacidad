1. Introducción

Biska POS es una aplicación diseñada para ayudar a pulperías, tiendas y pequeños negocios a gestionar ventas, inventario, clientes, compras, gastos y otros procesos administrativos.

Esta Política de Privacidad explica qué información puede recopilarse, cómo se utiliza y cómo protegemos los datos de nuestros usuarios.

2. Información que recopilamos

Dependiendo de las funciones utilizadas dentro de la aplicación, podemos recopilar y almacenar la siguiente información:

  • Nombre del negocio
  • Correo electrónico del usuario
  • Información de inicio de sesión
  • Productos e inventario
  • Clientes y proveedores
  • Ventas, compras y movimientos
  • Configuraciones del negocio
  • Información básica del dispositivo
  • Datos de uso y analíticas

Parte de esta información puede almacenarse de manera segura utilizando servicios de Firebase proporcionados por Google.

3. Servicios de terceros utilizados

Biska POS puede utilizar servicios de terceros para ofrecer funcionalidades específicas:

  • Firebase Authentication: para inicio de sesión seguro.
  • Cloud Firestore: para sincronización y almacenamiento de datos.
  • Firebase Cloud Messaging: para notificaciones push.
  • Firebase Analytics: para mejorar la experiencia del usuario y analizar el uso de la aplicación.
  • Google Sign-In: para autenticación mediante cuenta Google.
  • Apple Sign-In: cuando esté disponible en dispositivos compatibles.

Estos servicios pueden recopilar cierta información técnica conforme a las políticas de privacidad de Google y Apple.

4. Cómo utilizamos la información

La información recopilada se utiliza para:

  • Permitir el funcionamiento correcto de la aplicación
  • Sincronizar información entre dispositivos
  • Realizar respaldos en la nube
  • Mejorar rendimiento y estabilidad
  • Enviar notificaciones importantes
  • Brindar soporte técnico
  • Mejorar funciones y experiencia del usuario

5. Seguridad de la información

Implementamos medidas razonables de seguridad para proteger la información almacenada dentro de la plataforma.

Sin embargo, ningún sistema puede garantizar seguridad absoluta, por lo que recomendamos a los usuarios:

  • Utilizar contraseñas seguras
  • No compartir credenciales
  • Mantener actualizado su dispositivo
  • Realizar respaldos periódicos

6. Compartición de datos

No vendemos información personal de los usuarios.

La información únicamente puede compartirse con servicios tecnológicos necesarios para el funcionamiento de la aplicación, como Firebase o servicios de autenticación.

7. Derechos del usuario

El usuario puede solicitar:

  • Eliminar su cuenta
  • Eliminar datos almacenados
  • Consultar información relacionada
  • Solicitar soporte o aclaraciones

8. Cambios a esta política

Esta Política de Privacidad puede actualizarse ocasionalmente. Los cambios serán publicados en esta misma página.
